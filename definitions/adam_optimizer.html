Adam Optimizer

Adam (Adaptive Moment Estimation) is a popular optimization algorithm used to train deep learning models. It combines the benefits of two other methods: AdaGrad and RMSProp, adapting the learning rate for each parameter.

  • Maintains moving averages of both the gradients (first moment) and their squares (second moment).
  • Adapts the learning rate for each parameter, making training faster and more stable.
  • Widely used in training neural networks due to its efficiency and reliability.
Figure: Adam optimizer updates each parameter using estimates of the first and second moments of the gradients.

How It Works (Step-by-Step)

  1. Compute Gradients: Calculate the gradient of the loss with respect to each parameter.
  2. Update Moments: Update the moving averages of the gradients and their squares.
  3. Bias Correction: Correct the bias in the moment estimates.
  4. Parameter Update: Update each parameter using the adapted learning rate.

Code Example from microgpt.py

Below is a simplified excerpt from our training loop, showing the Adam optimizer update:

# Adam optimizer update
lr_t = learning_rate * (1 - step / num_steps)
for i, p in enumerate(params):
    m[i] = beta1 * m[i] + (1 - beta1) * p.grad
    v[i] = beta2 * v[i] + (1 - beta2) * p.grad ** 2
    m_hat = m[i] / (1 - beta1 ** (step + 1))
    v_hat = v[i] / (1 - beta2 ** (step + 1))
    p.data -= lr_t * m_hat / (v_hat ** 0.5 + eps_adam)
    p.grad = 0
Adam adapts the learning rate for each parameter, making training more robust and efficient.

Why Adam Optimizer Matters

  • Speeds up convergence and helps avoid local minima.
  • Reduces the need for manual learning rate tuning.
  • Is the default optimizer for many deep learning projects.
