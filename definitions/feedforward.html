Feed Forward Network

Feed forward networks are the basic building blocks of neural networks. In transformers, they process each position independently after attention, helping the model learn complex transformations.

  • Consist of one or more linear layers (matrix multiplications) and nonlinear activations (like ReLU).
  • Transform the input to a new representation, enabling the model to learn more abstract features.
  • In transformers, applied after self-attention to each token separately.
Figure: A feed forward network applies a series of linear and nonlinear transformations to each input vector.

How It Works (Step-by-Step)

  1. Input: Each token's vector from the attention layer is used as input.
  2. Linear Transformation: The input is multiplied by a weight matrix and a bias is added.
  3. Activation: A nonlinear function (like ReLU) is applied.
  4. Second Linear Transformation: The result is transformed again to produce the output.

Code Example from microgpt.py

Below is a simplified excerpt from our gpt() function, showing the feed forward block:

# 2) MLP block
x_residual = x
x = rmsnorm(x)
x = linear(x, state_dict[f'layer{li}.mlp_fc1'])
x = [xi.relu() for xi in x]
x = linear(x, state_dict[f'layer{li}.mlp_fc2'])
x = [a + b for a, b in zip(x, x_residual)]
The feed forward block applies two linear transformations with a ReLU activation in between, and adds a residual connection.

Why Feed Forward Networks Matter

  • Enable the model to learn complex, nonlinear relationships.
  • Increase the expressive power of each transformer layer.
  • Are a core component of all modern neural networks, not just transformers.
