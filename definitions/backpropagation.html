Backpropagation

Backpropagation is the algorithm used to compute gradients for all parameters in a neural network. It enables efficient training by applying the chain rule to propagate errors backward through the computation graph.

  • Starts from the loss (output) and works backward to compute gradients for each parameter.
  • Uses the chain rule of calculus to efficiently compute derivatives.
  • Is the foundation of training deep neural networks.
Figure: Backpropagation computes gradients by traversing the computation graph in reverse, from output to input.

How It Works (Step-by-Step)

  1. Forward Pass: Compute the output of the network and the loss.
  2. Backward Pass: Starting from the loss, recursively compute gradients for each node using the chain rule.
  3. Parameter Update: Use the computed gradients to update the model parameters.

Code Example from microgpt.py

Below is a simplified excerpt from our Value class, showing the backward method:

def backward(self):
    topo = []
    visited = set()
    def build_topo(v):
        if v not in visited:
            visited.add(v)
            for child in v._children:
                build_topo(child)
            topo.append(v)
    build_topo(self)
    self.grad = 1
    for v in reversed(topo):
        for child, local_grad in zip(v._children, v._local_grads):
            child.grad += local_grad * v.grad
Backpropagation traverses the computation graph in reverse order, applying the chain rule to compute gradients for every node.

Why Backpropagation Matters

  • Enables efficient training of deep neural networks.
  • Allows models to learn from data by adjusting parameters to minimize loss.
  • Is the backbone of all modern deep learning optimization.
