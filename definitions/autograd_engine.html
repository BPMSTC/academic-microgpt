Autograd Engine

Autograd (Automatic Differentiation) is the heart of modern deep learning frameworks. It automatically computes gradients (derivatives) for all operations in a computation graph, enabling neural networks to learn from data via backpropagation.

  • Tracks every mathematical operation performed on data (tensors/values) as nodes in a graph.
  • Builds a computation graph where each node represents a value or operation.
  • During the backward pass, it traverses this graph in reverse to compute gradients for each parameter, which are used to update the model.
  • Without autograd, training deep neural networks would be error-prone and extremely tedious.
Figure: Example of a computation graph for z = (x + y) * y. Each node stores its value and how it was computed. During backpropagation, gradients flow backward through the graph.

How It Works (Step-by-Step)

  1. Forward Pass: Compute the output by applying operations (e.g., addition, multiplication) and record how each value was produced. This builds the computation graph dynamically as the code runs.
  2. Backward Pass: Starting from the final output (usually the loss), recursively apply the chain rule to compute gradients for each node. This is done by traversing the graph in reverse order.
  3. Parameter Update: Use the gradients to update model parameters (weights) and improve performance. This is typically handled by an optimizer like Adam.

Code Example from microgpt.py

Below is a simplified excerpt from our actual Value class, which implements autograd in microgpt.py:

class Value:
    def __init__(self, data, children=(), local_grads=()):
        self.data = data
        self.grad = 0
        self._children = children
        self._local_grads = local_grads

    def __add__(self, other):
        other = other if isinstance(other, Value) else Value(other)
        return Value(self.data + other.data, (self, other), (1, 1))

    def __mul__(self, other):
        other = other if isinstance(other, Value) else Value(other)
        return Value(self.data * other.data, (self, other), (other.data, self.data))

    def backward(self):
        topo = []
        visited = set()
        def build_topo(v):
            if v not in visited:
                visited.add(v)
                for child in v._children:
                    build_topo(child)
                topo.append(v)
        build_topo(self)
        self.grad = 1
        for v in reversed(topo):
            for child, local_grad in zip(v._children, v._local_grads):
                child.grad += local_grad * v.grad
This code builds a computation graph as you perform operations, and then computes gradients for every value in the graph when you call backward().

Why Autograd Matters

  • Enables efficient and correct training of neural networks by automating gradient computation.
  • Reduces human error and makes experimentation with new architectures much easier and safer.
  • Forms the backbone of modern deep learning libraries (e.g., PyTorch, TensorFlow, and our own microgpt.py).
  • Allows for rapid prototyping and innovation in AI research and applications.

Try It Yourself!

Here's a minimal example you can run directly from our codebase:

# Forward pass
x = Value(2.0)
y = Value(3.0)
z = (x + y) * y

# Backward pass (compute gradients)
z.backward()
print(x.grad, y.grad)  # Gradients of z w.r.t x and y
This will print the gradients for x and y with respect to z, showing how autograd propagates derivatives backward through the computation graph.
